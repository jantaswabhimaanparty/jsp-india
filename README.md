# 🇮🇳 JSP India

## Janata Swabhimaan Party - Official Digital Platform

JSP India is the official digital platform of Janata Swabhimaan Party.

## Features

- Member Registration
- Digital Membership Card
- QR Verification
- Organization Management
- Citizen Services
- Janata Bazaar
- Health Services
- Education
- Farmer Services
- Jobs & Skill Development
- Events Management
- News Portal
- Complaint System
- Donation Management
- AI Assistant
- Admin Dashboard

## Technology Stack

Frontend:
- React
- Vite
- Tailwind CSS

Backend:
- Node.js
- Express.js

Database:
- PostgreSQL

Mobile:
- Flutter
